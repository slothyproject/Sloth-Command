FAQ

Answers for admins, mods, and owners

Fast answers for common setup and operations questions, so your team can keep moving when the server gets busy.

Do I need every permission enabled?

No. Start with required permissions and add extra access only for features you use.

Can I use Sloth Lee in a small server?

Yes. It works for small communities and scales with larger moderation teams.

How do I keep moderation actions consistent across mods?

Use role-based access and a shared escalation policy so warnings, mutes, and bans follow the same flow.

Where can I see what actions were taken and by whom?

Use battle logs and moderation history views to audit each action, actor, and timestamp.

What should I do during a raid spike?

Enable anti-raid presets, slow mode where needed, and lock high-risk channels until threat level drops.

Can I automate ticket workflows?

Yes. You can route ticket categories, assign staff, and standardize closing notes for repeatable handling.

How does API authentication work?

Use a bearer API key from the dashboard and rotate keys on a schedule for better operational hygiene.

Where do I report a setup issue?

Use your server support flow or the dashboard support channels so logs can be reviewed quickly.
