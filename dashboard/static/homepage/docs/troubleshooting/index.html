Troubleshooting

Use this page when something feels wrong and you want practical checks before opening a full support case.

Bot is online but commands fail

Re-sync slash commands, then verify the bot role is above target roles and has command permissions.

Moderation action says missing access

Check channel-level overrides and role hierarchy. Discord denies actions when role order is too low.

Tickets are not opening

Confirm ticket channels exist and the bot can create threads/messages in those channels.

Dashboard looks out of date

Refresh session, confirm current server selection, and check status page before deeper debugging.

Raid shield did not trigger

Review thresholds and cooldown values. Very high thresholds can delay protection activation.

Still stuck

Collect screenshots, command names, and timestamps, then contact support through your server workflow.
