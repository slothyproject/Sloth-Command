# Dissident

Dissident is a modular Discord bot for moderation, ticketing, logging, automod, leveling, economy, music, and more. Manage your entire community from an AI-powered dashboard or directly in chat.

---

## Quick Start

1. [Invite the bot](https://discord.com/oauth2/authorize) to your server
2. Run `!setup` to initialize default config
3. Open [slothlee.xyz/dashboard](https://slothlee.xyz/dashboard)
4. Connect your Discord account and manage settings from the web UI

## What You Can Do

- **Moderate:** Ban, kick, warn, mute, timeout, mass delete, slowmode
- **Ticketing:** Create support tickets with custom categories and transcripts
- **Logging:** Log message edits, deletions, joins, leaves, bans, voice
- **Automod:** Auto-delete invites, spam, caps, and custom regex rules
- **Leveling:** XP system with customizable roles and leaderboards
- **Economy:** Virtual currency, daily rewards, balance tracking
- **Music:** Play from YouTube, Spotify, or SoundCloud with queue control
- **AI Operator:** Ban, mute, edit channels, send messages — through natural language
